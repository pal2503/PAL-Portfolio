PalAcademy - Student Management System Architecture

Full Stack Architecture

Backend (Spring Boot)

  • JPA Entities: Model mapping of Student, Course, Enrollment (grade status), and User (credentials) with proper relational annotations (OneToMany, ManyToOne).
  • SHA-256 Hashing: Custom password encryption util implemented for security.
  • REST Controllers: Standardized JSON request handlers supporting pagination, validation filters, and cross-origin controls.
  • Data Initialization: Application runner class automatically loads baseline records into H2/MySQL on deployment for immediate sandbox testing.

Frontend (React Vite)

  • Vite Configuration: Proxy mappings for routing dynamic `/api/` triggers seamlessly to local server port.
  • State Tracking: Session caching utilizing localStorage persistence.
  • Component breakdown: Custom state directories comprising Auth views, Student directory listings, Catalog components, and grading modules.
  • Lucide Visuals: Vector icons providing a crisp vector aesthetics rendering instantly on dynamic grids.

Source files for this architecture exist locally in this computer inside the Projects/Java-FullStack-Project/student-management-system folder.
